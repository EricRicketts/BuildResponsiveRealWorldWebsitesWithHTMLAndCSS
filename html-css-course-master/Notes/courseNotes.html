My Notes for Section Three

Colors can be rendered by the RGB color model.

Each of the base colors can be represented by a value between 0 and 255. All three colors at maximum yield while, while all three colors at zero give black.

An RGB example is rgb(255, 0, 0) where the first value represents red, the second green, and the third blue. There is also a fourth value, which is transparent and is represented by a value between 0 and 1. The greater the number, the more transparent the color is. This means we can stack colors one and top of another and see through them.

Colors can be rendered by the Hexadecimal color model.

The hexadecimal example is #rrggbb, where a digit 0 to f is used in each of the red, green, and blue components. When a digit is repeated, it is equivalent to multiplying the value by 16. For example, #ff0000 is equivalent to #f00. This is called the shorthand notation.

In practice the hexadecimal notation is preferred over the rgb notation, but the rgb notation is used if we need transparency.

Gray is an important color spectrum, shades of gray appear when all three numbers are the same. For example, #808080 is a medium gray.
